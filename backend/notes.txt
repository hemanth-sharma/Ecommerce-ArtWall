Proposed API Endpoints
Home

GET /api/home: Returns data for the home page (e.g., featured products, banners).
All Products

GET /api/products: Returns a list of all products.
Filtering by Categories
GET /api/products?category=categoryName: Returns products filtered by category.
Single Product

GET /api/products/{id}: Returns details of a specific product by its ID.
Wishlist

GET /api/wishlist: Returns the user's wishlist items.
POST /api/wishlist: Adds a product to the user's wishlist.
DELETE /api/wishlist/{id}: Removes a product from the user's wishlist.
Cart

GET /api/cart: Returns the user's cart items.
POST /api/cart: Adds a product to the user's cart.
DELETE /api/cart/{id}: Removes a product from the user's cart.
PUT /api/cart/{id}: Updates the quantity of a specific item in the cart.
Authentication

Login
POST /api/login: Authenticates the user.
Signup
POST /api/signup: Registers a new user.
Search

GET /api/products/search?q=query
